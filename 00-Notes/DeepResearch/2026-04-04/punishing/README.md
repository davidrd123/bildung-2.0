# Punishing

This folder holds prompts that are mainly for pressure, contestability, overclaim audits, and method hardening.

They are **not** the default workflow for active translation phases.

Use them only when there is a specific reason to stop the work and ask harder structural questions:

- is a protocol overbuilt?
- is a frame doing too much?
- is a genealogy becoming a broad confirmer?
- is a model loop laundering fluent synthesis?
- is a design layer getting ahead of what the repo has earned?

If the actual task is:

- translate the next batch
- settle glossary pressure
- record real instability
- decide the next source-bound move

then stay out of this folder and use `productive-generative.md` instead.
