## What this sharpens (repo method)

This tranche is the most useful PSF surface so far for *translation discipline* precisely because it is not about “symbolic form” as a slogan. It is about how a medium can (or cannot yet) carry relation.

What it sharpens operationally:

- **Don’t treat word-meaning as primary.** The “sentence before word” claim (`0151` ~29-60) is a direct warning against dictionary-first translation when the unit of sense is syntactic connection. If the repo’s translation practice has a default unit, this is a pressure toward: start with clause relations before polishing lexical choices.
- **“Formless language” is a false category.** Cassirer uses isolating languages to show relation can live in word order (`0151` ~105-143). That maps cleanly onto a local discipline: if the source’s relation is carried by placement, the translator can’t “recover it” by adding neat connective words without cost. This is a residue/violence site.
- **Hypotaxis is earned.** The parataxis→hypotaxis arc (`0151` ~284-400) frames conjunctions / relatives / subordinate clauses as late instruments. That makes “clean logical connectives everywhere” a suspicious translation tendency: it can impose later relation-organs onto earlier/other relation-media.
- **Copula is not just logic.** The copula section (`0152` + `0153`) is a technical warning: “is” often starts as existence / place / time, and only later becomes the pure predicative link. This is a reason to keep `sein/ist`-family decisions under suspicion when a text is doing ontology, predication, or metaphysical pressure work.

## Practical use (if needed)

Use this note when a live translation/decryption problem looks like:

- the English is “right” lexically but relation is wrong (misattached clauses, incorrect subordination, forced connectives)
- a term feels stable but a later sentence reveals it was a relation-carrier
- “is” / “being” is doing more than predication (existential / locative / temporal residue)

If this note starts functioning as a general interpretive frame for everything, it’s being misused.
