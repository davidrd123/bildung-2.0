# Stereoscopic / parallax translation as candidate method move

Status: candidate method note. Not yet pressure-tested in live repo work. Not authority for translation practice in general.

Purpose: name a bounded move that the April 16 chat cluster articulated clearly enough to be worth testing, while keeping it below default-method status until one real repo trial earns it.

## Problem it sharpens

Some passages appear to be doing grammatical or morphological work that ordinary close translation and glossary pressure can feel but not fully name.

The question is whether a deliberately resistant multi-target pass can surface that hidden work more clearly.

## Candidate move

Run one source passage through two or more target languages chosen for *different* structural failure modes, then compare the strain.

The output is not the translations by themselves. The output is the `strain report`:

- what had to be added
- what could not be carried
- what the target overemphasized
- what the source seems to have been relying on silently

This is `stereoscopic` or `parallax` translation in the narrow sense: one target gives a flat image; the differences between targets give depth.

## What it is for

At best, the move can generate hypothesis-objects about:

- grammatical commitments
- morphological welding
- hidden register shifts
- inheritance pointers
- places where a current interpretation may depend on source-language structure more than the current notes yet show

## What it is not for

- not a replacement for close reading
- not a substitute for philological competence
- not evidence by itself
- not a default pass on every passage
- not permission to treat model-generated multilingual output as settled scholarship

## Current constraints

The move is only honest if all of the following are kept explicit:

1. The result is heuristic, not probative.
2. The passage is already known well enough that the output can be checked against existing understanding.
3. At least one real audit path exists:
   - specialist scholarship
   - target-language competence
   - source-context recheck
4. The result stays as hypothesis-pressure until that audit is run.

## Good first test

The best first test is not a new or unknown passage.

It is one passage already under real pressure where:

- the source language is clearly doing unusual work
- the repo already has notes strong enough to distinguish signal from fluent artifact
- success and failure are both legible

The April 16 chat's own best candidate was a Uexküll `Funktionskreis` / `Planmäßigkeit` passage, precisely because the repo already knows where the English strains.

## Failure modes

- If the target-language renderings are weak, the `strain reports` describe model limitation rather than source structure.
- If every passage yields equally interesting strain, the move is generating atmosphere rather than tracking real grammatical load.
- If the method gets used without pass metadata, the differences cannot be attributed and therefore do not count as data.
- If the move starts being used by convenience on every difficult passage, it becomes another way of widening the apparatus ahead of earned need.

## What would make it earn more

One real repo trial would need to show:

- some confirmed strain already sensed by existing notes
- at least one new-but-recognizable pressure point
- a clean negative condition for future use

That is, the trial should teach not only when the move helps, but when it does not.

## Relation to existing practice

This note does not introduce a foreign method so much as name a stronger version of something already present:

- open-term discipline preserves strain rather than resolving it away
- replay packets isolate one local pressure under explicit controls
- bounded survey encounters allow a scaffolded source to be touched once without opening a campaign

`Stereoscopic / parallax translation` would be one more bounded hypothesis-generation move, if it survives use.

## Current judgment

This is strong enough to keep in `method/` as a candidate because it sharpens a live repo problem that the current apparatus does not yet name well.

It is **not** strong enough yet to count as a standing repo method.

The first repo-honest low-amplitude trial is now on disk at:

- `00-Notes/experiments/2026-04-17/experiment-01-uexkuell-controlled-english-parallax-planmaessigkeit-outside-the-body.md`

That trial earned a small but real gain around the difference between methodological aspect and doctrinal hardening in `Planmaessigkeit`, but it did not justify broad adoption.

The next honest step is still another bounded experiment or a refusal, not broad adoption.
