## Productive morphemes

| Morpheme | Function | Examples on the page |
|---|---|---|
| `Vor-` | before, pre-, fore- | Vorurteil, Vorstellung, Voraussetzung, Vorgänger |
| `er-` | completion, achievement | Erkenntnis (← kennen), Erfahrung (← fahren), Erwerb (← werben), erheben, erschließen, erreichen |
| `be-` | directed at object, often transitive | Begriff (← greifen), Bedeutung (← deuten), Bewußtsein (← wußt + sein), Beobachtung, Behandlung, Beschränkung, Betrachtung, Beschaffenheit |
| `ver-` | transformation, completion-through; sometimes going-astray | Verstand (← stehen), Verfahren (← fahren), Vermögen (← mögen), Verständnis, verfälscht, vermischen |
| `-lich` | adjective from noun (= English `-ly`) | menschlich, freundlich, wissenschaftlich, wahrhaft (`-haft` cousin), geistlich (religious/clerical) |
| `-heit` / `-keit` | abstract noun from adjective (= English `-ness`/`-ity`); `-keit` after `-lich`/`-ig` | Wahrheit, Wirklichkeit, Notwendigkeit, Möglichkeit, Bedingtheit, Bestimmtheit |
| `-ung` | abstract noun from verb (= English `-tion`/`-ment`/`-ing`) | Wahrnehmung, Vorstellung, Erörterung, Behandlung, Beobachtung, Beschränkung, Bedeutung, Ausführung, Auslegung, Unterscheidung |
| `-isch` | adjective from Greek/Latin noun (= English `-ic`/`-ical`) | philosophisch, psychologisch, dialektisch, methodisch, empirisch, aphoristisch, metaphysisch, mathematisch |

Additional: `-bar` = "-able/-ible" (denkbar, gangbar, erkennbar). `-nis` = rarer abstract noun suffix (Erkenntnis, Verständnis).

The German prefix-and-suffix system is largely a calque of Latin philosophical morphology: most abstract nouns map morpheme-for-morpheme onto Latin equivalents (`Vor-aus-setz-ung` = `prae-sup-positio`, `Vor-stell-ung` ≈ `re-praesent-atio`, `Vor-urteil` = `prae-judicium`). Knowing Latin philosophical vocabulary unlocks German morphology mechanically.

## Grammar features

### Separable verbs

Many German verbs have a separable prefix that detaches in main clauses and reattaches in subordinate/infinitive forms.

- Infinitive / subordinate clause: prefix + verb glued. *entgegen-stehen, an-zünden, zurück-kommen, wahr-nehmen.*
- Main clause: conjugated verb at position 2, prefix at end of clause. *"Bacon kommt auf diesen Satz zurück"* = "Bacon returns to this proposition."
- Past participle: `-ge-` inserted *between* prefix and stem. *hin-zu-ge-füg-t* = added to.

Recognition rule: if a verb sits at position 2 and a weird adverb-like word appears at the end of the clause, try gluing them.

Examples from the page: `entgegenstehen, wiedergeben, anzünden, aufzeigen, hinzufügen, zurückkommen, festhalten.`

### Adjective endings

Adjectives change endings based on case, gender, number, and the preceding article. Two practical shortcuts:

- `-en` is the most common ending and is grammar, not meaning. When you see `-en` on an adjective, parse it as inflection.
- Chunk pattern: `des [adj]-en [noun]-s` = "of the [adj] [noun]" (genitive singular masculine/neuter). Example: `des menschlichen Vorstellens` = "of human representing." This chunk recurs constantly in philosophical German.

### Joining `-s-` in compound nouns

The *Fugen-s* connector carries no meaning; it just smooths the join between elements. Don't try to parse it as a morpheme. Examples: `Wahrheits-wert, Gattungs-charakter, Lieblings-satz, Bewußtseins-leben.`

### Negation

- `un-` prefix on adjectives/adverbs: `unmittelbar` (immediate, "un-means-able"), `unsicher, ungenau, unvollständig.`
- Combined with `-bar`: `unzerstörbar` (indestructible).

## Word families to recognize

### Directional prefixes (spatial cousins of `Vor-`)

| Prefix | Spatial sense | Examples |
|---|---|---|
| `auf-` | up, onto, open | Aufbau, Aufgabe, aufzeigen, Anbruch |
| `aus-` | out of | Auslegung, Ausführung |
| `ein-` | in, into | Eindruck, einzeln |
| `an-` | on, at, toward | Anschauung, Anfang, anrufen, anhaften, anzünden |
| `zu-` | to, closed, toward | Zugang, zurück, zufällig |
| `gegen-` | against, opposite | Gegenstand, Gegensatz, gegenüber, entgegenstehen |

Philosophical payoff: `Gegenstand` (object) = "the thing standing-against." The Latin *objectum* technically means the same ("thrown-against"), but English has lost the spatial metaphor. German keeps it alive.

### Anaphoric particles: `da-` / `hier-` / `wo-` + preposition

Workhorses of philosophical German prose. They reference back to a previously-mentioned abstract antecedent. English now has the equivalents (thereby, wherein, whereupon, therefrom) but they've narrowed to formal register; German speakers use these in everyday speech.

| Word | Decomposes as | English equivalent |
|---|---|---|
| `dadurch` | da- + durch | thereby, through it |
| `damit` | da- + mit | therewith, so that |
| `darin` | da- + in | therein |
| `daraus` | da- + aus | therefrom, from it |
| `daran` | da- + an | thereat, on it |
| `darüber` | da- + über | thereover, about it |
| `dazu` | da- + zu | thereto, to that |
| `hierin` | hier- + in | herein |
| `worauf` | wo- + auf | whereupon |
| `wovon` | wo- + von | whereof |

When you see `dadurch` or `damit` mid-paragraph, mentally substitute "thereby" or "through-it"; it's pointing back to the previous sentence's content.

### The cognition cluster

Cassirer uses these almost interchangeably in places, but they're *different metaphors*:

| Word | Root metaphor | Literal sense | English |
|---|---|---|---|
| `Verstand` | stehen (to stand) | the standing-through | understanding, intellect |
| `Vernunft` | vernehmen (to perceive/heed) | the heeded | reason |
| `Begriff` | greifen (to grip) | the grasped | concept |
| `Vorstellung` | stellen (to set/place) | the placing-before | representation, idea |
| `Anschauung` | schauen (to look) | the looking-at | intuition (Kant's sense) |
| `Wahrnehmung` | nehmen (to take) | the taking-as-true | perception |
| `Empfindung` | empfangen (to receive) | the received | sensation |
| `Erkenntnis` | kennen (to know) | the achieved-knowing | cognition |

Each is built from a different verb metaphor. When Cassirer distinguishes them, the morphology is doing argumentative work — the spatial-receptive-cognitive metaphors are structurally different operations of mind, not synonyms.

## Philosophically loaded compound decompositions

### Wirklichkeit (reality)

```
wirk + -lich + -keit
work/operate    adj.    abstract noun
```

`wirken` = to operate, to take effect. `wirklich` = operative. `Wirklichkeit` = the state of being operative.

Reality in German is *what-is-working* — preserves Aristotle's *energeia* in a way English's "reality" (from Latin `res` = thing) does not. Caveat: the metaphor is *morphologically available* but mostly sleeping in everyday use; philosophical writers (Hegel, Heidegger, Cassirer) reactivate it deliberately.

### Tatsache (fact)

```
Tat + Sache
deed   thing
```

A fact is "a deed-thing" — the deposited result of doing. Identical etymology to English `fact` (from Latin *factum* = done thing), but English has lost the etymology while German keeps it transparent.

### Vernunft (reason)

```
Ver- + nunft
       (archaic perfect of nehmen, "to take")
```

Reason in German is "the heeded" / "the perceived" — structurally identical to Greek *logos* (what is said/heard) and parallel to Latin *ratio* (calculation). Each tradition's word for reason carries a different metaphor.

### Wahrnehmung (perception)

```
Wahr + nehm + -ung
true    take    noun
```

`wahrnehmen` (separable verb) = literally "to take-as-true." Perception in German has a judgment already baked in — it's not neutral receiving but active taking-as-true. Compare `Empfindung` (sensation, "the received") for the contrast.

### Begriff (concept)

```
Be- + griff
      (root: grasp, from greifen)
```

A concept in German is "the grasped" — what mind has gripped. The metaphor is alive because `greifen` is still everyday vocabulary (compare English "comprehend" from *com-prehendere* = grasp-together, but English speakers don't feel the grasp anymore).

### Maßstab (standard, measure)

```
Maß + Stab
measure  rod
```

A standard is a "measure-rod" — the surveyor's tool. When Bacon writes that sense is a `schwankender und ungenauer Maßstab`, the physical metaphor of the wavering measuring-stick is present in the word.

### unmittelbar (immediate)

```
un- + mittel + -bar
not    means      -able
```

"Un-means-able" → without intermediary → immediate. English "immediate" (Latin *im-mediatus*) has the same etymology but it's buried.

### Voraussetzung (presupposition)

```
Vor- + aus- + setz + -ung
before  out    set    noun
```

"Pre-out-setting" — what is set-forth in advance. Exact morpheme-for-morpheme mapping to Latin *prae-sup-positio*.

## What's missing

Deliberately. This note covers vocabulary-level morphology but not:

- Case system (nominative / accusative / dative / genitive) and how it shapes article and adjective endings beyond the `des [adj]-en [noun]-s` chunk
- Subordinate-clause word order (verb-final after `daß`, `weil`, `wenn`, relative pronouns, etc.) — the single feature that most disrupts English-speaker parsing
- Modal particles (`doch`, `denn`, `ja`, `mal`, `eben`) which carry register and emphasis without translatable lexical content
- Periodic sentence structure with suspended verb at clause end, which is where Cassirer's prose is hardest to read

These are the natural next areas if the morpheme work has stabilized.
