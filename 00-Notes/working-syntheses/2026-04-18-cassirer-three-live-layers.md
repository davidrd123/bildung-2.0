# Cassirer — Three Live Layers

Status: working-layer integration note. Not a whole-author synthesis. Not a
license to collapse current Cassirer pressure into one frame.

## Why this note exists

The repo now has enough Cassirer on disk that "Cassirer" no longer names one
undifferentiated pressure.

There are at least three distinct live layers:

1. the completed `Erkenntnisproblem` / `Zeitmauer` operational layer
2. the source-earned `Substanzbegriff` systematic layer
3. the localized but still reserve-pressure `Philosophie der symbolischen
   Formen` layer

This note exists to keep those layers from being conflated.

## Layer 1 — `Erkenntnisproblem` / operational Cassirer

Main surface:

- [cassirer-pressure-on-zeitmauer.md](/Users/daviddickinson/Projects/Lora/bildung-2.0/texts/zeitmauer/threads/cassirer-pressure-on-zeitmauer.md:1)

What this layer gives:

- epistemic-historical constitution of objectivity
- measure as constitutive before it becomes flattening
- anti-reification pressure on `Weltplan`, `Urgrund`, and related hardenings
- naming crisis is not license for anti-concept drift

What it does **not** give:

- apparatus-world
- organism-world
- symbolic forms as active repo architecture
- a positive theory of qualitative time

Use this layer when the question is:

- how exact science historically secures objectivity
- how a methodological gain hardens into metaphysical misuse
- how Jünger's pressure on measure should be kept from becoming naive
  anti-science

## Layer 2 — `Substanzbegriff` / systematic source-earned Cassirer

Main surfaces:

- [cassirer-substanzbegriff-und-funktionsbegriff/README.md](/Users/daviddickinson/Projects/Lora/bildung-2.0/sources/german/cassirer-substanzbegriff-und-funktionsbegriff/README.md:1)
- [2026-04-17-substanzbegriff-bounded-survey-against-zeitmauer-cassirer-pressure.md](/Users/daviddickinson/Projects/Lora/bildung-2.0/00-Notes/working-syntheses/2026-04-17-substanzbegriff-bounded-survey-against-zeitmauer-cassirer-pressure.md:1)

What this layer adds beyond `Erkenntnisproblem`:

- experimental and inductive science are constructive from the start
- objectivity is graded stability under criticism, not fixed outside
- projection gives way to selection and apperceptive ordering
- relation is not only a historical outcome but a systematic principle of
  object-constitution

What it does **not** give:

- symbolic-forms architecture
- language as a distinct mode of world-constitution
- a general license to read every cross-project problem through Cassirer

Use this layer when the question is:

- how scientific objectivity is built
- how relation replaces substance in a stronger systematic sense
- how ordered worlds can be constituted without naive givenness or metaphysical
  projection

Current status:

- this is the present **source-earned Cassirer center**
- it has paid rent
- it still does not justify a live `texts/` campaign

What this layer gives so far:

- language as constitutive objectivation
- symbolic form as a mode of geistige Energie, not a loose framework
- world-view difference internal to language, not just different labels on one
  ready-made world
- chapter V now sharpens this layer further: it is the Cassirer surface for
  medium-specific relation-expression, not only for language in general

What it does **not** yet give:

- a full symbolic-forms campaign
- a stable symbolic-forms glossary
- a repo-wide right to invoke `symbolic form`
- authorization to treat Volumes II and III as equally live

Use this layer when the question is:

- whether a problem requires Cassirer specifically at the level of language as
  formative medium
- whether objectivity is being constituted through symbolic articulation rather
  than only through scientific method
- whether current work is reaching for symbolic forms too loosely and needs a
  direct source check

Current status:

- this layer is real
- it is no longer merely chat-atmospheric
- it is still **reserve pressure**, not center

## Practical decision rule

If the live question is mainly about:

- epistemic-historical constitution, use `Erkenntnisproblem`
- constructive objectivity, use `Substanzbegriff`
- language as world-forming objectivation, use `PSF I`

If a note or chat just says "Cassirer" without this discrimination, the
pressure is still too vague.

## What remains held

- `Philosophie der symbolischen Formen` Volume II
- `Philosophie der symbolischen Formen` Volume III as active surface
- whole-author claims about Cassirer
- repo-wide symbolic-forms vocabulary import
- any collapse of Cassirer / Uexküll / Anders / Jünger into one single
  predecision theory

## Provisional judgment

The repo no longer needs to decide "Cassirer yes or no."

It needs to decide which Cassirer layer a given pressure actually belongs to.

Right now:

- `Substanzbegriff` is the strongest source-earned layer
- `PSF I` is the most important reserve layer
- the next worthwhile push should test whether `PSF I` adds something the
  current `Substanzbegriff` line cannot already carry
