## First Orientation

This is not one chat topic. It is a long working session around the startup of
`texts/erkenntnisproblem-vol2/`, with four interwoven strands:

- **Vol. II apparatus calibration:** audit Vol. I, check what Codex had already
  scaffolded, import only earned lessons, add hooks / structural-center /
  routing discipline, and test whether the work can move faster after a
  correctness gate is proven.
- **Operational verification:** review Parts 001-002, Latin footnotes, the
  Bacon chapter close, then the Gassendi chapter close. This is the
  source-proximate part of the session.
- **German morphology pedagogy:** use the Bacon opening to teach productive
  morphemes and grammar mechanics, then generalize cautiously toward
  translation as recovery of cognitive resources.
- **Meta-method and export:** discuss negative archaeology, Lotmanian productive
  asymmetry, Peirce/Lotman triadic differences, what to leave as calibration
  rather than teammate-legible artifact, and how to export Claude Code JSONL.

## Working Forms Discovered

1. **Correctness Gate Before Forward Motion.** Part 001 had to pass corrected
   German, substantive footnote treatment, draft English, opening finding,
   routing decision, and hooks before further motion. The gate caught a real
   Latin idiom error (`mente captus`) and then held through Part 002.

2. **Threshold of Trust.** Verification can ease only after several tranches
   pass the gate without substantive review corrections. The Bacon close and
   Gassendi close made this operational rather than rhetorical.

3. **Re-entry Hooks as Anti-Compression.** First-pass translation may stay
   provisional, but unresolved pressure has to be visible. Silent smoothing is
   the failure mode.

4. **Journal as Routing Surface.** The Zeitmauer comparison led to the Vol. II
   journal prompts: structural center, routing decision, re-entry hooks, and
   selective "what to expect next" at part boundaries.

5. **Meta Material as Pointer, Not Artifact.** The session explicitly decided
   that much of the Lotman / Peirce / Spengler / cognitive-infrastructure talk
   calibrated judgment without needing to become teammate-legible synthesis.

## Caution Zones

- Do not treat Claude Code product details from this transcript as stable
  documentation. Remote Control / rewind behavior was discussed inside a live
  product session and needs current docs if reused.
- Do not promote the political/civilizational material. It is calibration and
  pointer material unless later source work forces it.
- Do not use LLM introspection claims as authority. The session itself marks
  them as functional/metaphorical and confabulation-prone.
- Do not generalize "parallel translation" into a default method. The Lotman
  reading points to selective non-isomorphic rendering at high-pressure hooks,
  not multiplication of outputs everywhere.
- Do not let the Baconian gate become a checklist that fills itself. The gate
  works only while someone is bringing actual attention.
