## Working Form Discovered

The useful new form is **full-resolution traversal**.

Use it when the chat is a live reading operation in parts and premature classification would flatten what happened. A traversal records:

- line anchor;
- source locus;
- question being asked;
- provisional recovery;
- terms under pressure;
- cross-links;
- hold/verify status.

Only after that should the material be classified as source commentary candidate, comparative dossier, method note, actual question, verification queue, or sift residue.

## Phrases / Operations Worth Keeping

- recover before naming; name before promoting; verify before canonizing;
- the mode of attention this material calls for and rewards;
- trust enough to be wrong;
- foothold versus ultimacy claim;
- good compass, poor engine;
- cached symbolic use versus inquiry-mode;
- tight source text as forcing function;
- source-near-ish is not source authority;
- modernized Japanese is not original/classical Japanese.
