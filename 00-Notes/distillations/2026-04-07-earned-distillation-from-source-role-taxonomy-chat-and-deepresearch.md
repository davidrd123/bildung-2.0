## Promoted claims

### 1. A four-part source-role distinction is useful in the living layer

The chat sharpened a distinction that is better than `primary vs. secondary` for current work:

1. `primary sources` - the texts being translated, decrypted, or closely read
2. `source-language commentary` - formed readers working in the source language
3. `continued primary work` - later texts continuing the same problem-line rather than commenting on it
4. `thinker-on-thinker encounters` - strong minds metabolizing earlier strong minds into new work

This should be used as a reading filter, not yet as crystallized architecture.

### 2. Category 3 was missing as a name, not as a practice

The most important correction to the first pass is simple:

- `sources/modern/` already functions as the home of category 3
- what was missing was the genealogical description, not the encounter discipline

The repo already knows how to handle these texts:

- they are encountered as primary modern concept sources
- they stay `open`
- they enter only when live work triggers them
- they must teach something unexpected before they earn transfer authority

So the real gain here is descriptive clarity. It becomes easier to say what Rosen, von Foerster, Montévil/Mossio, McCulloch, or Eilenberg-Mac Lane are doing in the repo and why they are not merely imported frameworks.

### 3. The April 6 DeepResearch folder splits by source role more cleanly than by file title

The useful first cut is:

- `Secondary_Sources_*` = mostly category 2
- `Thinker_on_Thinker_*` = mostly category 4

But the cleaner point is that the files are not role-pure.

Examples:

- category 2 inside the secondary files: Koslowski on `Zeitmauer`, Schöttker's `Adnoten`, Takhtajan on `Tektologiya`, Volkening's reading diary, Ferrari on `Erkenntnisproblem`
- category 4 inside the thinker-on-thinker files: Heidegger, Plessner, Canguilhem, Merleau-Ponty, and Cassirer on Uexkuell; Heidegger, Blumenberg, and Blanchot on Juenger; Koyre as the main mediating node in the Cassirer -> Kuhn line; Hoesle on Gomez Davila
- category 3 pressure appearing at the margins: biosemiotics as a live Uexkuell descendant line, Louie/Hofmeyr as Rosen continuers, Deacon and Cariani as stronger Pattee continuers, and similar modern continuation lines

This matters because the research set should not be judged only by bibliographic accuracy. It should also be judged by what kind of encounter each item would actually support.

### 4. Source-language commentary fits existing repo practice without new infrastructure

The chat's best practical point survives intact:

- source-language commentary can be encountered under the same bounded-source discipline as other entries in `sources/`
- the encounter is with a formed reader rather than only with the original text
- this does not require a new object type

This is probably the most immediately useful result from the whole discussion.

### 5. Thinker-on-thinker material is real, but it should stay separate from ordinary commentary

The strongest contribution of the `Thinker_on_Thinker_*` work is not that it found better secondary sources. It found a different source role:

- not commentary that clarifies the source
- not modern continuation in the same technical lineage
- but a later mind turning the source into new production

This is the closest thing here to `Auseinandersetzung`.

That category is worth preserving because it changes what one is reading for. One does not read Dewey on Leibniz, Heidegger on Uexkuell, or Blumenberg on Juenger in order to get a safer summary of the earlier thinker. One reads them to see what the source became under pressure from another serious mind.
