## Promoted claims

### 1. The volume-1 result is now a portable mechanism-language

The main gain from `tektologiya-vol1` is no longer only sequential coverage. The project now has working English for a small mechanism-set:

- `conjugation`
- `ingression`
- `deingression`
- `selection`
- `mobile equilibrium`
- `law of leasts`
- `analytic sum`

This matters because the vocabulary was not imported from systems-history summary. It was forced through the text section by section, so each term now carries the residue of the passages that tested it.

### 2. `Tektologiya` is a loaded instrument for `bildung-2.0`, not yet a cross-project frame

Bogdanov is the most explicit local source for the claim that problems become more soluble when re-posed organizationally. That makes the subproject unusually dangerous in a productive way. Once the vocabulary stabilizes, it begins pressing toward use beyond its home text.

The correct description is therefore:

- `Tektologiya` is available as an instrument
- it is not yet licensed as a standing framework for the other projects

### 3. The rule has to be: wait until the other text asks for it

This is the main disciplinary point.

The tektological vocabulary should cross into other subprojects only when:

- the local source has already generated an organizational difficulty or boundary-problem of its own
- the Bogdanov term clarifies that difficulty rather than merely redescribing it
- the comparison remains reversible, meaning the other source can still resist it and push back

Otherwise the move is premature framework-import.

### 4. The best next uses are bounded experiments

Three experiments now look earned:

1. **Rosen encounter**
   This is the strongest next modern-source test because `organization`, `closure`, and the conditions of system-preservation are already explicit on both sides.
2. **`fused / beaded` internal test**
   The one still-tentative pair can be pressure-tested on the repo's own ecology or workspace structure without pretending it has already earned cross-project authority.
3. **Mechanism diagram**
   A compact visual map of the main tektological primitives and transitions would likely help future work more than another prose summary.

## Held below promotion

The following should stay live but unpromoted:

- treating Jünger's `Zeitmauer` directly as a tektological crisis-boundary
- treating Gómez Dávila's fragment-form as simply `beaded` in the Bogdanov sense
- treating Cassirer and Bogdanov as already aligned on the portability of knowledge before the Cassirer work locally generates the same problem
- treating the modern systems line as though Bogdanov had already been put into direct working contact with Rosen, autopoiesis, or second-order cybernetics inside the repo

These may become real. They are not yet repo findings.

## Practical use

Use this note when:

- choosing between more `Tektologiya` linear expansion and a bounded external comparison
- planning visual work around the mechanism-set
