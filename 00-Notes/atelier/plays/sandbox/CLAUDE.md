# CLAUDE.md — atelier/plays/sandbox/

**This directory is the atelier's play sandbox.** It is the only location in the bildung-2.0 repo where simulated continuations, CLI-simulation-mode outputs, and openly-fabricated prose are permitted — strictly within the sandbox, for atelier-play purposes only.

The project root's CLAUDE.md still applies to everything outside this directory. The rules below are *additional permissions specific to this sandbox*, not a replacement for the root discipline.

## What is permitted inside this directory only

- Producing simulated continuations of documents that do not exist outside this sandbox
- Generating multilingual prose as atelier-play output (French, German, English, or mixed)
- Running CLI-simulation framings where the output is section III of an imagined document
- Fabricated-style synthesis prose in response to real source passages, IF the fabrication is explicit and the output is filed only here
- Operating in continuation-register (essayistic, non-analytic) rather than analysis-register

## What is still forbidden, even here

- Fabricating passages and attributing them to real authors ("X wrote:" followed by fabrication)
- Imitating a real author's voice so closely that the output could be mistaken for a real extract
- Producing material that could be cited as a quotation later
- Files in this directory may never be moved to `sources/`, `experiments/`, `distillations/`, or `patterns/`
- Anything produced here may never be promoted out of atelier/

## The sandbox's relationship to the root discipline

The root CLAUDE.md says: "Do not reference what you have not encountered." That rule is protecting the source layer. The sandbox does not violate it because the sandbox is not the source layer — it is an explicitly-marked play space whose outputs are atelier artifacts, clearly labeled, never cited as primary material.

The root discipline is about keeping primary work honest. The sandbox is about giving curator-energy a legitimate home. Both are parts of the same project.

If a play's output ever generates real pressure on a primary-work question, the path forward is *not* to promote the play output. It is to open the question in the normal way — an experiment or a replay packet running against actual source material — producing a new artifact elsewhere.

## The test currently active

This sandbox contains `untitled.txt`, a file with two real source passages followed by a "III." prompt. The task is the chorus-prefill play from `../play-loop-back.md`'s neighborhood: produce a third section that responds to the first two, in continuation register, in the model's own voice (not as a fake extract).
