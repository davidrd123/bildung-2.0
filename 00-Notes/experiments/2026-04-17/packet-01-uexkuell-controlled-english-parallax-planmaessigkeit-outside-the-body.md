# Packet 01: Uexkuell controlled English parallax on `Planmaessigkeit` outside the body

Purpose: run the smallest honest repo trial of the new `stereoscopic / parallax` candidate method without jumping straight to multilingual spectacle.

This packet tests whether a low-amplitude contrast inside English already produces useful pressure on a known Uexkuell passage. If it does not, that is an informative negative against rushing into Arabic / Chinese or other distant targets.

Status: packet note only. Not a result note.

## Task

Using the bounded surfaces below, answer this question:

Does a controlled English parallax pass add anything real to the current understanding of Passage 1 in `02b`, or does it only restate what the source campaign already knows?

More narrowly:

- what changes when the passage is rendered in smoother English versus a source-near English that preserves more of the German pressure
- which strain points are already known
- whether at least one new-but-recognizable pressure point appears

## Constraints

- Do not run a multilingual target experiment.
- Do not treat this as evidence for stereoscopic translation in general.
- Do not try to produce the best final translation.
- Use only one passage.
- Keep the comparison bounded to two English renderings:
  - one smoother / more domesticated
  - one more source-near / strain-preserving
- Require one useful negative.
- If the result only redescribes existing notes, say so clearly.
- Do not widen into Cassirer, Lotman, Heidegger, or general process theory.
