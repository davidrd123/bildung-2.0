# Diagrams: Experiments 01–07

Purpose: visual representations of the experimental findings. Not canonical architecture. Use for fast orientation to what the experiments established.

Status note: the Bruno gap columns are findings (they make the type-mismatch visible in a way that changes understanding). The reification gradient is a finding (it shows the discriminator works). The Bayle-pass diagram is a finding (it shows which synthesis moves survived and which didn't). The experiment map is illustration (navigation, not new content). The "same shape" box carries a question mark because the experiment illuminated, not proved, the structural isomorphism.
