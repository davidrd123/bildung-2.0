# Bronowski - *The Origins of Knowledge and Imagination* - Text-Only Scaffold

Jacob Bronowski, *The Origins of Knowledge and Imagination*.

## What This Is

A split, navigable scaffold of *The Origins of Knowledge and Imagination* for
chapter-scale browsing in web chats and lightweight agent work.

This exists so the book can be touched directly, in bounded pieces, without
loading the full EPUB text into a single conversational turn.

This is **not** yet a live modern encounter, **not** a normalized working text,
and **not** a license to import Bronowski's framework into current text work as
if it had already been tested.

## Status

- The authority source on the shelf is the EPUB in
  `sources/modern/incoming/books/`.
- The current full text was generated with Calibre `ebook-convert`.
- The files under `source/raw/` are suitable for orientation, bounded direct
  forays, and web-chat access.
- The EPUB remains the authority whenever formatting, punctuation, page
  anchors, or notes matter.
- No journal, glossary, encounter dossier, or bounded pressure note has yet
  been earned from this scaffold alone.

## Why Here

- Not only `sources/modern/incoming/books/` because the raw EPUB shelf is not
  usable for chapter-scale browsing.
- Not `texts/bronowski-origins-knowledge-imagination/` because no campaign,
  glossary, journal, or encounter apparatus has been earned.
- `sources/modern/` is the least misleading place for a modern English source
  being localized for possible later use without promotion into a live
  encounter.

## How To Use

- Read `source/raw/*.txt` when you want foreword- or chapter-scale access.
- Use `source/sections.yaml` when you need sequence, title, and extracted line
  range.
- Treat this as a browsing scaffold, not as a quotation-grade critical text.

If the book later earns real pressure, the next honest move would be a bounded
encounter note or a checked working tranche for whichever chapter is actually
being pressed.
