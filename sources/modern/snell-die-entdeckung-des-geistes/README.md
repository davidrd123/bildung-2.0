# Snell — *Die Entdeckung des Geistes* — Text-Only Scaffold

Bruno Snell, *Die Entdeckung des Geistes: Studien zur Entstehung des
europäischen Denkens bei den Griechen* (Vandenhoeck & Ruprecht, 9. Auflage,
2009).

## What This Is

A split, navigable scaffold of *Die Entdeckung des Geistes* for chapter-scale
browsing in web chats and lightweight agent work.

This exists so the book can be touched directly, in bounded pieces, without
loading the full PDF into a single conversational turn.

This is **not** yet a live modern encounter and **not** a license to import
Snell's frame into current text work as if it had already been tested. It now
has a machine-normalized OCR layer for search and reading, but the text still
needs witness-backed repair before exact quotation.

It does now have one live pressure bridge: alongside Cassirer PSF II, this
scaffold activates the mythic Greek substrate shelf at
`../../greek/mythic-greek-cassirer-snell-substrate/README.md`.

The chapter-level intersection and the Snell/PSF II gaps are mapped in:

- `../../../00-Notes/working-syntheses/2026-05-19-snell-psf-ii-intersection-and-gaps.md`

## How To Use

- Read `source/normalized/*.txt` when you want introduction- or chapter-scale
  access.
- Use `source/raw/*.txt` when you need the extraction witness beneath the
  normalized layer.
- Use `source/sections.yaml` when you need the sequence, title, extracted line
  range, or printed page start.
- Treat this as a browsing scaffold, not as a quotation-grade critical text.
- Follow `ocr-cleanup-plan.md` for the manual repair sequence. The PSF II live
  overlap starts with Homer, Olympian gods, Hesiod, early lyric, tragedy,
  human/divine knowledge, and historical consciousness.

If the book later earns deeper pressure beyond this Greek substrate bridge, the
next honest move would be a bounded encounter note or a checked working tranche
for whichever chapter is actually being pressed.
