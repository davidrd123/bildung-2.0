# Snell — *Die Entdeckung des Geistes* — Text-Only Scaffold

Bruno Snell, *Die Entdeckung des Geistes: Studien zur Entstehung des
europäischen Denkens bei den Griechen* (Vandenhoeck & Ruprecht, 9. Auflage,
2009).

## What This Is

A split, navigable scaffold of *Die Entdeckung des Geistes* for chapter-scale
browsing in web chats and lightweight agent work.

This exists so the book can be touched directly, in bounded pieces, without
loading the full PDF into a single conversational turn.

This is **not** yet a live modern encounter, **not** a normalized working text,
and **not** a license to import Snell's frame into current text work as if it
had already been tested.

## Status

- The source PDF has an OCR text layer from ABBYY, so the extraction is useful
  but should be treated as OCR-derived.
- The files under `source/raw/` are suitable for orientation, bounded direct
  forays, and web-chat access.
- The PDF remains the authority whenever Greek, punctuation, page transitions,
  or exact quotation matter.
- No journal, glossary, encounter dossier, or bounded pressure note has yet
  been earned from this scaffold alone.

## Why Here

- Not only `sources/modern/incoming/books/` — because the raw PDF shelf is not
  usable for chapter-scale browsing.
- Not `texts/snell-die-entdeckung-des-geistes/` — because no campaign,
  glossary, journal, or encounter apparatus has been earned.
- `sources/modern/` is the least misleading place for a modern scholarly
  interlocutor being localized for possible later use without yet promoting it
  into a live encounter.

## How To Use

- Read `source/raw/*.txt` when you want introduction- or chapter-scale access.
- Use `source/sections.yaml` when you need the sequence, title, extracted line
  range, or printed page start.
- Treat this as a browsing scaffold, not as a quotation-grade critical text.

If the book later earns real pressure, the next honest move would be a bounded
encounter note or a checked working tranche for whichever chapter is actually
being pressed.
