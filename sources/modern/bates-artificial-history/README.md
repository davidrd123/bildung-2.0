# Bates — *An Artificial History of Natural Intelligence* — Text-Only Scaffold

David W. Bates, *An Artificial History of Natural Intelligence* (University of
Chicago Press, 2024).

## What This Is

A split, navigable scaffold of *An Artificial History of Natural Intelligence*
for chapter-scale browsing in web chats and lightweight agent work.

This exists so the book can be touched directly, in bounded pieces, without
loading the full PDF into a single conversational turn.

This is **not** yet a live modern encounter, **not** a normalized working text,
and **not** a license to import Bates's framework into current text work as if
it had already been tested.

Reserve note on disk:

- `sources/modern/incoming/2026-04-19-bates-artificial-history-under-pressure.md`
  records why the source is being localized, which chapter clusters are under
  pressure, and what this scaffold still does not license. It now also records
  bounded reads of chapters `17-18` and `20-21 / 25-27`: Bates is more useful
  as a reserve interlocutor on exteriorization, error, and exosomatic
  transmission than as a Uexkull interpreter.

## Status

- The source here comes from a born-digital PDF, so the extraction is cleaner
  than the OCR-derived scaffolds.
- The files under `source/raw/` are suitable for orientation, bounded direct
  forays, and web-chat access.
- Raw chapter files are numbered directly by chapter (`001`-`029`) so chapter
  references and filenames now line up cleanly.
- The PDF remains the authority whenever formatting, punctuation, or page
  transitions matter.
- No journal, glossary, encounter dossier, or bounded pressure note has yet
  been earned from this scaffold alone.

## Why Here

- Not only `sources/modern/incoming/books/` — because the raw PDF shelf is not
  usable for chapter-scale browsing.
- Not `texts/bates-artificial-history/` — because no campaign, glossary,
  journal, or encounter apparatus has been earned.
- `sources/modern/` is the least misleading place for a contemporary English
  interlocutor whose text is being localized for possible later use without yet
  promoting it into a live encounter.

## How To Use

- Read `source/raw/*.txt` when you want chapter-scale access.
- Use `source/sections.yaml` when you need the sequence, title, and extracted
  line range.
- Treat this as a browsing scaffold, not as a quotation-grade critical text.

If the book later earns real pressure, the next honest move would be a bounded
encounter note or a checked working tranche for whichever chapter is actually
being pressed.
