# Weatherby — *Language Machines* — Text-Only Scaffold

Leif Weatherby, *Language Machines: Cultural AI and the End of Remainder
Humanism* (University of Minnesota Press, 2025).

## What This Is

A split, navigable scaffold of *Language Machines* for chapter-scale browsing in
web chats and lightweight agent work.

This exists so the book can be touched directly, in bounded pieces, without
loading the full PDF into a single conversational turn.

This is **not** yet a live modern encounter, **not** a normalized working text,
and **not** a license to import Weatherby's framework into current text work as
if it had already been tested.

First bounded result on disk:

- `sources/modern/incoming/2026-04-19-weatherby-language-machines-under-pressure.md`
