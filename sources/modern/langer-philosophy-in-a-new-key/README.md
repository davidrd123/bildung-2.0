# Langer — *Philosophy in a New Key* — Text-Only Scaffold

Susanne K. Langer, *Philosophy in a New Key: A Study in the Symbolism of
Reason, Rite, and Art*.

## What This Is

A split, navigable scaffold of *Philosophy in a New Key* for chapter-scale
browsing in web chats and lightweight agent work.

This exists so the book can be touched directly, in bounded pieces, without
loading the full source into a single conversational turn.

This is **not** yet a live modern encounter, **not** a normalized working
text, and **not** a license to import Langer's framework into current text
work as if it had already been tested.

First bounded result on disk:

- `sources/modern/incoming/2026-04-19-langer-philosophy-in-a-new-key-under-pressure.md`
  now records grounded bounded contact on chapters 1-10 (the book’s full
  argument, including the rite/myth chapters). The result is source-real but
  still narrow: Langer has earned a localized modern pressure position on
  problem-framing as technique (question-form constrains thought), symbolization
  as a generative idea for the human sciences (including a critique of imported
  “militant methodology”), sign / symbol distinction, meaning as relational
  function, denotation as a tetradic pattern rather than a simple pointing
  relation, a symbolic-first account of language that resists
  communication-first reduction, a presentational-form claim that widens
  semantics beyond discursive language without collapsing into irrationalism,
  a rite/myth line (ritual as articulated gesture; myth as serious world-picture
  and forerunner of metaphysics), a music-centered account of presentational
  meaning as morphology of feeling rather than fixed denotation, a
  generalization from music into artistic import + “artistic truth” as adequacy
  rather than propositional paraphrase, and (in chapter 10) a tight integration
  of “fact” and “reality” as woven from signs and symbols (including charged
  symbols and disorientation as modern symbolic pathologies). If she is touched
  again, it should be for a concrete local pressure and should go through the
  authority PDF for the specific passage.

- The PDF remains the authority whenever formatting, punctuation, or page
  transitions matter.

## Why Here

- Not only `sources/modern/incoming/books/` — because the shelf pair is not
  usable for chapter-scale browsing.
- Not `texts/langer-philosophy-in-a-new-key/` — because no campaign, glossary,
  journal, or encounter apparatus has been earned.
- `sources/modern/` is the least misleading place for a modern English source
  being localized for possible later use without promotion into a live
  encounter.

## How To Use

- Read `source/raw/*.txt` when you want preface- or chapter-scale access.
- Use `source/sections.yaml` when you need the sequence, title, and extracted
  line range.
- Use `source/cleaned/*.txt` for current clean-text work. Check
  `source/cleaned/page-proof-manifest.md` before treating any span as
  image-proofed; printed pp. 76-77 are Kindle-screenshot word-proofed but not
  supplied-PDF-proofed, and the second-edition preface remains raw-witness-only
  unless another image witness is supplied.
- Run `python3 tools/verify_langer_philosophy_in_a_new_key.py` after changing
  `source/cleaned/` to recheck page-marker coverage, chapter footnote
  continuity, known OCR garbage patterns, and manifest/audit blocker notes.

If the book later earns real pressure, the next honest move would be a bounded
encounter note or a checked working tranche for whichever chapter is actually
being pressed.
