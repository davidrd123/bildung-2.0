# Peirce Encounter Notes

This directory holds bounded source-contact notes for the tracked
*Essential Peirce, Volume 1* scaffold.

It is not a full Peirce campaign. Use these notes when a specific local
question needs a small Peirce surface, and keep the source-status boundary
visible. The current whole-volume text layer is useful for reading, but most
sections remain pending image-grounded proof.
