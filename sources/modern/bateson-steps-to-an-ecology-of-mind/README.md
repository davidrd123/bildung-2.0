# Bateson - *Steps to an Ecology of Mind* - Text-Only Scaffold

Gregory Bateson, *Steps to an Ecology of Mind*.

## What This Is

A split, navigable scaffold of *Steps to an Ecology of Mind* for essay-scale
browsing in web chats and lightweight agent work.

This exists so the book can be touched directly, in bounded pieces, without
loading the full source into a single conversational turn.

This is **not** yet a live modern encounter, **not** a normalized working text,
and **not** a license to import Bateson's framework into current text work as
if it had already been tested.

## Status

- The current split scaffold is built from the TXT witness on the incoming
  shelf.
- No authority PDF has been localized in this source directory.
- The files under `source/raw/` remain useful as a browsing scaffold and for
  bounded source contact, but they are not quotation-grade.
- The TXT witness does not preserve reliable printed page starts for section
  boundaries, so `source/sections.yaml` intentionally leaves
  `printed_page_start` as `null`.
- The localized full text is generated locally under `source/full/`, which is
  gitignored.
- The witness heading for `The Roots of Ecological Crisis` appears as `The
  Boots of Ecological Crisis`; the metadata follows the contents listing while
  preserving the raw witness text.

## Why Here

- Not only `sources/modern/incoming/books/` - because the shelf text is not
  usable for essay-scale browsing.
- Not `texts/bateson-steps-to-an-ecology-of-mind/` - because no campaign,
  glossary, journal, or encounter apparatus has been earned.
- `sources/modern/` is the least misleading place for a modern English source
  being localized for possible later use without promotion into a live
  encounter.
