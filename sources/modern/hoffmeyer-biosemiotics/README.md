# Hoffmeyer — *Biosemiotics* — Text-Only Scaffold

Jesper Hoffmeyer, *Biosemiotics: An Examination into the Signs of Life and the
Life of Signs*.

## What This Is

A split, navigable scaffold of *Biosemiotics* for chapter-scale browsing in web
chats and lightweight agent work.

This exists so the book can be touched directly, in bounded pieces, without
loading the full PDF into a single conversational turn.

This is **not** yet a live modern encounter, **not** a normalized working text,
and **not** a license to import Hoffmeyer's framework into current text work as
if it had already been tested.

## Status

- The authority source on the shelf is the PDF in
  `sources/modern/incoming/books/`.
- The current split scaffold is built from the companion TXT witness extracted
  from that PDF.
- The files under `source/raw/` are suitable for orientation, bounded direct
  forays, and web-chat access.
- The extraction is OCR / text-layer noisy in the front matter, at some chapter
  title pages, and in the late back-cover scan material.
- The PDF remains the authority whenever formatting, punctuation, page
  transitions, or doubtful words matter.

## Why Here

- Not only `sources/modern/incoming/books/` — because the shelf pair is not
  usable for chapter-scale browsing.
- Not `texts/hoffmeyer-biosemiotics/` — because no campaign, glossary,
  journal, or encounter apparatus has been earned.
- `sources/modern/` is the least misleading place for a modern source being
  localized for possible later use without promotion into a live encounter.

## How To Use

- Read `source/raw/*.txt` when you want chapter-scale access.
- Use `source/sections.yaml` when you need the sequence, title, and extracted
  line range.
- Treat this as a browsing scaffold, not as a quotation-grade critical text.

If the book later earns real pressure, the next honest move would be bounded
cleanup of the actually-pressed chapter files against the authority PDF, not a
premature campaign buildout.
