# Stengers -- *Thinking with Whitehead* -- Text-Only Scaffold

Isabelle Stengers, *Thinking with Whitehead: A Free and Wild Creation of
Concepts*, translated by Michael Chase, with a foreword by Bruno Latour
(Harvard University Press, 2011).

## What This Is

A split, navigable scaffold of *Thinking with Whitehead* for chapter-scale
browsing in web chats and lightweight agent work.

This exists so the book can be touched directly, in bounded pieces, without
loading the full PDF into a single conversational turn.

This is **not** yet a live modern encounter, **not** a normalized working text,
and **not** a license to import Stengers's Whitehead reading into current text
work as if it had already been tested.

## Status

- The source here comes from the OCR/text layer of the PDF on the incoming
  books shelf.
- The extraction is readable but visibly noisy in places, especially around
  spacing, ligatures, running headers, and some numerals.
- The files under `source/raw/` are suitable for orientation, bounded direct
  forays, and web-chat access.
- The PDF remains the authority whenever formatting, punctuation, or page
  transitions matter.
- No journal, glossary, encounter dossier, or bounded pressure note has yet
  been earned from this scaffold alone.

## Why Here

- Not only `sources/modern/incoming/books/` -- because the raw PDF shelf is not
  usable for chapter-scale browsing.
- Not `texts/stengers-thinking-with-whitehead/` -- because no campaign,
  glossary, journal, or encounter apparatus has been earned.
- `sources/modern/` is the least misleading place for a modern English source
  being localized for possible later use without promotion into a live
  encounter.

## How To Use

- Read `source/raw/*.txt` when you want foreword-, introduction-, chapter-, or
  conclusion-scale access.
- Use `source/sections.yaml` when you need the sequence, title, and extracted
  line range.
- Use `reader/` for bootstrap orientation before asking a web chat to work
  through the raw text.
- Treat this as a browsing scaffold, not as a quotation-grade critical text.

If the book later earns real pressure, the next honest move would be a bounded
encounter note or a checked working tranche for whichever chapter is actually
being pressed.
