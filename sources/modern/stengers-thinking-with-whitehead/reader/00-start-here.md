# Start Here

This reader layer is a bootstrap aid for reading Stengers's *Thinking with
Whitehead* in a cloned repo or web chat. It is not a commentary campaign, not a
normalized edition, and not an earned project synthesis.

The authority order is simple:

1. The PDF on the incoming shelf remains the authority for exact wording.
2. `source/raw/*.txt` is the chapter-scale text witness for chat use.
3. `reader/*` is only a set of orientation tools for first-pass understanding.

## Fast Route

For a first pass, do not ask a chat to read the full book. Start with this set:

1. `reader/01-whitehead-minimal-primer.md`
2. `reader/02-stengers-reading-strategy.md`
3. `reader/glossary.yaml`
4. `source/raw/010-introduction-whitehead-today.txt`
5. `reader/chapter-guides/010-introduction-whitehead-today.md`

Then move through:

1. `source/raw/021-the-mathematician-and-the-sunset.txt`
2. `reader/chapter-guides/021-the-mathematician-and-the-sunset.md`
3. `source/raw/027-interlude-a-pragmatics-of-concepts.txt`
4. `reader/chapter-guides/027-interlude-a-pragmatics-of-concepts.md`

That gives a first grip on the problem before the book becomes technical.

## Better First-Pass Sequence

After the three starting pieces above, a reasonable route through the whole
book is:

1. Introduction
2. Chapters 1-2
3. Chapter 7
4. Chapters 8, 13, 14
5. Chapters 16-18
6. Chapters 23-25
7. Conclusion

This route is not a replacement for the book. It is a way to avoid getting
stranded before the conceptual machinery has started to make sense.

## External Orientation

The Stanford Encyclopedia of Philosophy entry on Whitehead is a useful outside
map for the basic chronology and the broad labels: mathematics and logic,
philosophy of science, metaphysics, process philosophy.

- <https://plato.stanford.edu/entries/whitehead/>

Use it for orientation only. Stengers is not writing an SEP-style overview. She
is trying to show how Whitehead's concepts work under pressure.

## What To Ask A Chat

Use chapter-scale prompts. Good first questions:

- What problem is this chapter trying to make unavoidable?
- Which Whitehead term is being introduced, and what is it supposed to prevent?
- Where is Stengers refusing a standard philosophical question?
- What would I misunderstand if I treated this as ordinary metaphysics?
- What should I carry into the next chapter?

Avoid asking for a total summary too early. In this book, the useful unit is
often the question-form, not the conclusion.

## Guardrails

- Do not treat the glossary as settled Whitehead scholarship.
- Do not treat Stengers as a neutral explainer of Whitehead.
- Do not quote from the OCR text when exact wording matters; check the PDF.
- Do not import terms into `bildung-2.0` project vocabulary unless later source
  work creates a concrete need.
