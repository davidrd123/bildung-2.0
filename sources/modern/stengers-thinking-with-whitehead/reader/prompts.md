# Prompt Pack

These prompts are written for a cloned-repo workflow. Replace the chapter paths
as needed.

## First Session

```text
Read:

- sources/modern/stengers-thinking-with-whitehead/reader/00-start-here.md
- sources/modern/stengers-thinking-with-whitehead/reader/01-whitehead-minimal-primer.md
- sources/modern/stengers-thinking-with-whitehead/reader/02-stengers-reading-strategy.md
- sources/modern/stengers-thinking-with-whitehead/reader/glossary.yaml
- sources/modern/stengers-thinking-with-whitehead/source/raw/010-introduction-whitehead-today.txt
- sources/modern/stengers-thinking-with-whitehead/reader/chapter-guides/010-introduction-whitehead-today.md

Explain the introduction for a reader who knows only the SEP-level basics of
Whitehead. Do not summarize paragraph by paragraph. Reconstruct the problem
Stengers is setting up and the reading posture she wants.
```

## Chapter Walkthrough

```text
Read the chapter and its guide:

- <chapter-path>
- <guide-path>

Give me:

1. The chapter's guiding question.
2. The false problem or easy dualism Stengers is resisting.
3. The key Whiteheadian concept being tested.
4. The argument path in 6-10 steps.
5. What I should carry into the next chapter.
```

## Concept Clarifier

```text
Using the glossary and the current chapter, explain <concept> in three layers:

1. Plain-English first grip.
2. What work the concept does in this chapter.
3. What misunderstanding it is designed to prevent.

Do not present the concept as settled beyond the chapter unless the text earns
that extension.
```

## Confusion Debugger

```text
I am stuck on this passage/chapter. Diagnose the confusion by separating:

1. OCR noise or extraction noise.
2. Missing Whitehead background.
3. Stengers's deliberate shift in question.
4. A genuinely unresolved conceptual difficulty.

Then suggest the next two paragraphs or sections to reread.
```

## Build A Reading Ledger

```text
After reading this chapter, create a compact reading ledger with:

- concepts introduced or pressured
- unresolved terms
- passages to check against the PDF before quotation
- links to earlier or later chapters in sections.yaml
- one question to bring forward
```

## Project-Contact Check

```text
Does this chapter create any concrete pressure on existing bildung-2.0 work?

Be conservative. Distinguish:

- direct source pressure
- loose resonance
- vocabulary that should remain unpromoted
- possible future encounter note
```
