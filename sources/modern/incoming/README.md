# Modern Incoming

Purpose: hold modern-source material before it has earned promotion into a
top-level `sources/modern/` encounter or a full book scaffold.

This directory is not one object type. It is an intake area with separate
lanes. Keep the lanes explicit so raw shelf material, pressure notes, local
scratch files, and article references do not collapse into one queue.

## Lanes

- `books/`: raw book shelf. Mostly ignored in git; source scaffolds point back
  here for their authority files.
- `local/`: local-only scratch, SyncThing handoffs, and private working
  sources. Ignored in git.
- `under-pressure/`: dated pressure notes. These preserve why a source became
  methodologically visible without promoting it into an encounter.
- `papers/`: scientific-paper intake, tracked source PDFs, raw reference
  extractions, and control-case support files.
- `peirce/`, `ferrari/`, `grothendieck/`: small tracked source surfaces for
  author-specific extracts or first-contact material. Do not generalize this
  into a `curated/` hierarchy until repeated pressure forces it.
- `acquisition-list/`: acquisition packets and commentary shelves. Incoming
  survey only.
- `provisional/`: unclassified reserve PDFs or sources that have not yet
  earned a clearer lane.

Loose root files are legacy or one-off intake items. Normalize them only when
work actually touches them; do not reorganize by symmetry alone.

## Intermediate Source Status

The notes here often use `encounter_status` in frontmatter because that was the
early convention. Read it as source status, not as a claim that an encounter has
already happened.

Useful intermediate statuses:

- `located`: source is on disk or reachable, and the reason to preserve it is
  recorded. No substantial source contact is implied.
- `first-contact`: a narrow touch verified or corrected the pressure.
- `bounded-contact`: selected passages, chapters, or sections were read for a
  specific pressure, without opening a campaign.
- `control-case`: a source tests a live claim from outside the source's own
  tradition or vocabulary.
- `scaffolded`: a split browsing surface exists, but it is still not a live
  encounter or campaign.
- `superseded`: retained for provenance; a newer note carries the current
  pressure.

Promotion into a top-level encounter requires source contact that changes how
live work is read. Topical relevance is not enough.
