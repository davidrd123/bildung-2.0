# Under Pressure

Purpose: preserve source pressure that is real enough to record but not earned
enough to become a top-level modern encounter.

These notes are not a queue. They are traces of moments when a source became
methodologically visible enough to keep in view without importing its framework
into live work.

## What Belongs Here

- a source-localized reason to keep a text near the project
- a first-contact note that verifies or weakens a candidate pressure
- a bounded-contact note from selected passages or chapters
- a control-case note that tests a live claim from outside its home vocabulary
- a superseded note retained for provenance

## What Does Not Belong Here

- raw PDFs or EPUBs
- full book scaffolds
- cleaned working texts
- promoted encounter notes
- instructions that create work by merely existing

## Status Vocabulary

- `located`: source identified and reason preserved; no substantial read.
- `first-contact`: one focused touch; pressure checked at small scale.
- `bounded-contact`: selected source surface read under explicit limits.
- `control-case`: external comparison point for a claim already under test.
- `superseded`: old pressure state retained; use the newer linked note.

If a note needs more than these statuses, the source probably needs a local
README or a promoted encounter rather than more intake taxonomy.
