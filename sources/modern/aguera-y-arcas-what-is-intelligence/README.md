# Agüera y Arcas — *What Is Intelligence?* — Text-Only Scaffold

Blaise Agüera y Arcas, *What Is Intelligence?: Lessons from AI About
Evolution, Computing, and Minds*.

## What This Is

A split, navigable scaffold of *What Is Intelligence?* for chapter-scale
browsing in web chats and lightweight agent work.

This exists so the book can be touched directly, in bounded pieces, without
loading the full source into a single conversational turn.

This is **not** yet a live modern encounter, **not** a normalized working text,
and **not** a license to import Agüera y Arcas's framework into current text
work as if it had already been tested.

## Status

- The current split scaffold is built from the local PDF witness on the
  incoming shelf.
- The PDF text layer is usable for browsing extraction.
- The files under `source/raw/` remain useful as a browsing scaffold and for
  bounded source contact, but they are not quotation-grade.
- The extractor removes a repeated source footer line before writing the
  localized text.
- `source/sections.yaml` records `pdf_page_start` for each section, but printed
  page numbers are not reliable in this e-book-derived PDF and are left as
  `null`.

## Why Here

- Not only `sources/modern/incoming/books/` — because the shelf PDF is not
  usable for chapter-scale browsing.
- Not `texts/aguera-y-arcas-what-is-intelligence/` — because no campaign,
  glossary, journal, or encounter apparatus has been earned.
- `sources/modern/` is the least misleading place for a modern English source
  being localized for possible later use without promotion into a live
  encounter.
