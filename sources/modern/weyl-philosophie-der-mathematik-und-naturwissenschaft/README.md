# Weyl - *Philosophie der Mathematik und Naturwissenschaft* - OCR Scaffold

Hermann Weyl, *Philosophie der Mathematik und Naturwissenschaft*, 8. Auflage.

## What This Is

A page-level localization scaffold for OCR, direct page inspection, and later
chapter-scale cleanup.

This is **not** yet a cleaned German working text, **not** a chapter split, and
**not** a live encounter apparatus. The current PDF is primarily image-only, so
the first useful layer is page images plus page-level bookkeeping.

## Status

- The current scaffold is built from the local PDF witness on the incoming
  book shelf.
- The embedded text layer is usable only for the publisher/title pages.
- The body of the book is image-only and needs OCR plus direct visual checking.
- `source/pages.yaml` records every PDF page and whether `pdftotext` found
  embedded text on that page.
- `source/text-layer/pages/*.txt` contains only pages where an embedded text
  layer exists.
- Generated full text, page images, and OCR outputs are local-only and
  gitignored.
- Tesseract is installed, but the German language pack is not currently
  available in this environment. Installed languages are `eng`, `osd`, and
  `rus`.

## Why Here

- Not only `sources/modern/incoming/books/` - because the PDF is not usable for
  bounded page inspection or OCR correction as-is.
- Not `texts/weyl-philosophie-der-mathematik-und-naturwissenschaft/` - because
  no cleaned text, glossary, journal, or translation/encounter apparatus has
  been earned.
- `sources/modern/` is the least misleading place for a modern source being
  localized for possible later work without promotion into a live campaign.

## Workflow

Refresh the embedded text-layer manifest:

```bash
python3 tools/scaffold_weyl_philosophie_der_mathematik.py
```

Extract page images for direct inspection:

```bash
python3 tools/scaffold_weyl_philosophie_der_mathematik.py --images --dpi 200 --format jpeg
```

After installing German Tesseract data, run OCR over the generated page images:

```bash
python3 tools/scaffold_weyl_philosophie_der_mathematik.py --skip-text --ocr --ocr-lang deu
```

For a smaller visual/OCR batch, use `--first` and `--last`.
