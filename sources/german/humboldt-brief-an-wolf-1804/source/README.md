# Source Layout

This directory holds the local witness and extracted text layers for
Humboldt's 16 June 1804 letter to Friedrich August Wolf.

## Local Witnesses

- `local/` - gitignored IA text PDF and OCR sidecars.
- `raw/` - tracked raw letter and targeted passage extracts.
- `normalized/` - checked working tranche for the exact Cassirer-facing
  `Sprachstudium` passage.

The IA image-container PDF and processed JP2 zip were not landed. The text PDF
is sufficient for this small support shelf; if citation-critical collation is
needed later, download the visual witness from IA item `10604215bsb`.
