# Source Notes

This source layer holds local witnesses and extraction surfaces for Humboldt's
GS IV *Ueber das vergleichende Sprachstudium in Beziehung auf die
verschiedenen Epochen der Sprachentwicklung*.

## Witness Policy

- `source/local/gesammelteschrif04humbuoft.pdf` is the working text PDF.
- `source/page-images/jp2/` is the visual witness from the original JP2 tar.
- hOCR, page-index, search-text, scan-data, and page-number sidecars are
  locating aids.
- `source/full/` and `source/raw/` are OCR/PDF text surfaces. They are
  searchable and useful for substrate work, but remain uncorrected.
- `source/normalized/` should contain only checked tranches that earned use in
  an actual reading or cross-text move.

## Current Local Artifacts

All files under `source/local/` and `source/page-images/` are gitignored.
The tracked authority surface is the metadata plus `full/` and `raw/`.
