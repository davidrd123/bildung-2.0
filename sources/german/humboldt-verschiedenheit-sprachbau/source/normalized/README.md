# Normalized Extracts

This directory is for checked working tranches derived from `../raw/`.

Normalization may repair OCR noise, join broken words, or regularize obvious
extraction artifacts, but it should not harmonize witnesses silently. Record
important variants, uncertain readings, and page anchors in the file itself or
in `../page-map.yaml`.
