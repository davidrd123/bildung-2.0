# Kant — *Kritik der Urteilskraft* — Localized Source Scaffold

Immanuel Kant, *Kritik der Urteilskraft*.

## What This Is

A localized German source scaffold for narrow forays into the third Critique. It exists so that conversational instances, web chats, and future agents can read specific source passages directly in German without loading the full text at once.

This is **not** yet a live Kant encounter and **not** a promoted subproject. The pressure note remains at:

- `sources/modern/incoming/2026-04-15-kant-third-critique-under-pressure-from-uexkuell.md`

That note records why the source is under pressure from the Uexküll campaign. This directory only localizes the text itself.

## Why Here

- Not only in `sources/modern/incoming/` — because the pressure note belongs there, but the source text itself is a German primary text and should live on the source-language side.
- Not yet a full source campaign like `sources/german/uexkuell-theoretische-biologie/` — because no glossary, journal, or encounter dossiers have been earned for Kant.

So the current split is deliberate:

- `sources/modern/incoming/...` holds the pressure note
- `sources/german/kant-kritik-der-urteilskraft/` holds the localized source scaffold

## Split Logic

This is a lightweight navigational split, not yet a scholarly edition.

The current segmentation is:

- front matter / Inhaltsübersicht
- Vorrede
- Einleitung split by Kant's Roman-numeral divisions I–IX
- the body split at the actual section starts of the work as they appear in the text
- short container files preserved where useful (`Erster Teil`, `Erste Abteilung`, `Anhang`, etc.)

This gives a practical browsing granularity for exploratory source contact, especially around the teleological-judgment side (`§§ 61–91`).

## How To Use It

- Use `source/full/` when you need the whole work in one file.
- Use `source/raw/` when you want bounded source forays at section scale.
- Use `source/sections.yaml` to find the teleological tranche quickly, map a split file back to full-text line numbers, or hand off exact loci to another agent.
