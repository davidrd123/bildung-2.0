# Neoplatonic, Epicurean, and Skeptic Cassirer Substrate

**Status:** Cassirer substrate shelf. This is a targeted search surface, not a
standing campaign for Plotinus, Epicureanism, or skepticism.

## Why This Exists

Plotinus is pressured through Renaissance and symbolic-form material. Epicurus
is pressured through Gassendi and the empiricism line. Sextus/Pyrrhonism is a
watch surface for the skepticism material. Diogenes Laertius is included
because Book X is the practical ancient witness for Epicurus and because the
same text helps locate doxographic material for the Presocratic dossier.

## Source Shape

- `source/tei/` contains TEI XML fetched directly from PerseusDL or
  OpenGreekAndLatin/First1KGreek.
- `source/plain/` contains generated plain-text search surfaces.
- `source/manifest.yaml` records CTS URNs, source URLs, and edition notes.
- `source/checksums.sha256` records the local file checksums.

Use the TEI for citation-sensitive work. Use the generated plain text for
searching and fast triangulation while reading Cassirer.
