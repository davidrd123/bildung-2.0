# Presocratic and Atomist Doxography Cassirer Substrate

**Status:** Cassirer substrate shelf. This is a Diels-style search surface for
Presocratic and atomist pressure, not a clean fragment edition and not a full
Presocratic campaign.

## Why This Exists

Cassirer pressures Parmenides, Zeno, Heraclitus, Empedocles, Anaxagoras,
Democritus, Leucippus, Xenophanes, and the Pythagorean line through the PSF
apparatus, `Substanzbegriff`, and the `Erkenntnisproblem` Galilei/atomism
material.

The proper working unit is a Diels-style dossier organized by author and
fragment. The immediately curlable public surfaces are OCR scans, not clean
TEI. That makes this folder useful for discovery and search, while still
requiring verification against scans or a critical edition before citation.

## Source Shape

- `source/ocr/` contains Internet Archive DjVu OCR for Diels' `Fragmente der
  Vorsokratiker` and `Doxographi Graeci`.
- `source/manifest.yaml` records source URLs and quality notes.
- `source/checksums.sha256` records the local file checksums.

The TEI Diogenes Laertius shelf in
`../neoplatonic-epicurean-skeptic-cassirer-substrate/` is also relevant as a
doxographic witness, especially for practical author lookup.
