# Sources — Original-Language Encounters

## What This Is

A practice log, not a translation project. When a reference in the Escolios, the Zeitmauer, or the Exegesis points back to an original-language source, follow it here. The encounter is small: a passage in the original, an attempt at it, comparison with existing translations, notes on what the original language *does* that English can't.

## Why

Translation is formation, not information. Existing English translations of Seneca, Nietzsche, or Sophocles are perfectly adequate as products. But the *act* of sitting with a sentence in its original language — feeling why this word and not another, discovering what the grammar encodes that no translation reproduces — changes the reader. The value is in the practice, not the output.

## How Entries Are Triggered

Not by a syllabus. By following references encountered in the three reading/translation projects:

- An escolio quotes or parodies a Latin formula → follow it back
- A Jünger section cites a Greek concept → look at the Greek
- A PKD entry references Heraclitus or Parmenides → find the fragment

The thread index below records what triggered each encounter.

## Structure

```
greek/          # Homer, Sophocles, Plato, Diogenes Laertius, Heraclitus...
latin/          # Seneca, Petrarch, Aquinas, liturgical formulae...
german/         # Nietzsche, Hegel, Hölderlin... (beyond the Zeitmauer project)
french/         # La Rochefoucauld, Valéry, Montaigne, Tocqueville...
```

Each file is one encounter: a passage, an attempt, notes. Files are named by author and reference, not numbered.
