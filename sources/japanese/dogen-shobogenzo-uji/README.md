# Dōgen — *Shōbōgenzō* 有時 (Uji) — Text Scaffold

道元禅師『正法眼蔵』第十一巻「有時」

## What This Is

A text scaffold of *Uji* (Being-Time), fascicle 11 of the *Shōbōgenzō*, extracted
from a modern Japanese (現代語訳) rendering.

This is **not** the original classical Japanese/kanbun text. It is a modern
Japanese reading version with editorial glosses in angle brackets (〈〉) clarifying
implicit logical connectives.
