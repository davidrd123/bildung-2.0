# Lotman Reading Order

Working note for choosing which Ю. М. Лотман texts to bring into this repo and
which ones are best suited for cleaned text / audiobook preparation.

## Core Sequence

1. **Внутри мыслящих миров. Человек - текст - семиосфера - история**
   - English title: *Universe of the Mind*.
   - Current status: this is the Lotman book already scaffolded and used here.
   - Working judgment: best first Lotman for this project because it holds
     together text, semiosphere, and history in one systematic argument.

2. **Культура и взрыв**
   - Best next Lotman for this project.
   - Focus: discontinuity, explosion, unpredictability, historical dynamics.
   - Current status: DjVu source imported locally and initial cleanup scaffold
     created under `lotman-kultura-i-vzryv/`.

3. **Непредсказуемые механизмы культуры**
   - Late, dictated, and more accessible.
   - Useful if we want Lotman on unpredictability without the full systematic
     build of *Внутри мыслящих миров* or *Культура и взрыв*.

4. **Структура художественного текста**
   - Foundational literary-semiotic work.
   - More technical and literary than the current theoretical need.

## Other Important Lotman Texts

- **Анализ поэтического текста. Структура стиха**
  - Focused on poetry and textual analysis.

- **Семиотика кино и проблемы киноэстетики**
  - Narrower than the core sequence, but relevant if the project turns toward
    image/media semiotics.

- **Роман А. С. Пушкина «Евгений Онегин»: Комментарий**
  - Not general theory, but possibly Lotman's best example of
    cultural-contextual commentary as method.

- **Александр Сергеевич Пушкин. Биография писателя**
  - Literary-historical Lotman; less central for the theoretical spine.

- **Беседы о русской культуре / Лекции по русской культуре**
  - Very readable Russian culture material and a strong audiobook candidate.
  - More historical-cultural than systematic.

## Current Recommendation

For this repo's Lotman sequence:

1. Finish **Внутри мыслящих миров** access layers already in progress.
2. Clean **Культура и взрыв** page by page as the next major Russian Lotman
   source.
3. Bring in **Непредсказуемые механизмы культуры** if we want a more accessible
   late Lotman treatment of unpredictability.
4. Defer **Структура художественного текста** until the project needs a deeper
   literary-semiotic foundation.
