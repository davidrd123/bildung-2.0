# Encounters

Encounters are source-grounding passages tied to specific references in `texts/erkenntnisproblem-vol2`. Each encounter file includes corrected Latin, a working translation, and notes on how the source passage bears on Cassirer's reading.

Format (mirroring the Gassendi convention):

- **Trigger** — the Cassirer file, footnote, and citation that motivated the encounter
- **Witnesses** — which scans / PDFs / JP2 pages contain the passage
- **Local Context** — the passage's setting in Spinoza's own text
- **Cassirer Passage** — the Latin Cassirer quotes (corrected against the chosen witness)
- **Working Translation** — English rendering
- **Surrounding Argument** — what Spinoza is actually doing in the source, including material Cassirer did not quote
- **Notes** — how the source passage confirms, sharpens, or pressures Cassirer's reading
- **Re-entry** — pointers for future work

Longer sequential translations should move into `parts/` only if this campaign grows beyond citation-level grounding.

The natural first encounter is the *Ethica* I prop. 10 scholium, cited by Cassirer in Erk. Vol. 2 Part 030 footnote 2.
