# Parts

This directory is reserved for longer sequential translation chunks if the campaign grows beyond citation-level encounters.

It should remain empty until specific Vol. 2 pressure actually calls for a sequential walk-through of a passage longer than what fits an `encounters/` file (compare the Gassendi campaign: encounters live in `encounters/`, parts cover multi-page sequential walks through the *Dubitatio*/*Instantia* structure).

For the *Ethica* the natural unit if parts open would be by Proposition with its Demonstration / Scholium / Corollaries, since each Proposition is bounded enough to translate as one chunk and short enough to fit Cassirer-anchored verification.
