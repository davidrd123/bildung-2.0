# Raw Extracts

Raw outputs of the ABBYY XML page extractor live here. These preserve block typing (`[BODY]`, `[MARGINALIA]`, `[HEADER]`) and the original OCR character stream including the long-s artifact (`ſ` rendered as `f`).

Do not edit these by hand. Corrections belong in `../normalized/`, which records the long-s repair pass and any other resolution.
