# Parts

This directory keeps bounded transcription-and-translation slices for the Campanella *Metaphysica* source campaign.

The purpose is substrate-building, not promotion to `texts/`: make the Latin accurate enough, translated enough, and indexed enough that later chats can work directly with Campanella rather than with Cassirer's paraphrase alone.

## Part Shape

Each part should normally include the following when the material warrants it; the list is a guardrail, not a mold:

- source location: JP2 leaf, printed page when known, and section title if visible
- source status: raw extract path, normalized extract path, and verification caveats
- opening finding: what this slice makes newly usable
- corrected Latin, preserving meaningful uncertainty rather than silently smoothing it
- working translation
- notes on terms, OCR issues, and relation to Cassirer or adjacent Campanella passages
- translation risk: at least one lexical, contextual, edition, or residual difficulty
- re-entry: the next concrete place to continue

## Current Use

Use `parts/` for ordinary substrate slices. Use `encounters/` only when a passage needs citation-grade treatment as a Campanella/Cassirer anchor or a concept dossier.

## Numbering Rule

Reserve numeric `NNN-*` filenames for literal page-order transcription/translation. Page-order does not mean stopping mechanically at a physical page edge: when a leaf break cuts a sentence, subsection, or local argument, the numeric part should follow the Latin far enough to complete that unit and record the crossover. Non-sequential concept-chain work should use descriptive `pilot-*` filenames or move elsewhere before page-order work begins.

When page-order work starts in a non-contiguous region, use a local prefix that names the lane. The Part I trial keeps the global `001-*` name; the Part III reconnaissance uses `part-iii-NNN-*` so `part-iii-001` does not imply sequential continuation from the Part I trial.

## Pilot Graduation

`pilot-*` means non-sequential, not disposable. A pilot stays a pilot when it is a warranted concept-chain slice outside literal page order. It can graduate in only two ways:

- to an `encounters/` file if it becomes a citation-grade anchor or concept dossier needing full page-image collation and stable reference
- into numeric `NNN-*` page-order work if the campaign switches to literal sequential transcription and the surrounding pages are opened in order

For concrete non-promotion decisions applying this rule, see `../reading/cassirer-vol1-self-consciousness-crosswalk.md` and `../reading/cassirer-vol2-part018-intrinsecation-crosswalk.md`: after direct anchors are opened, promotion still depends on whether the useful artifact is a standalone encounter or a composite dossier.

## Slice Progression

For non-sequential work, name the warrant in prose before opening a pilot. Do not turn this into a fixed taxonomy or ranked queue.

Useful warrants are usually Cassirer's primary citations, a term distribution generated by the Latin, or a dependency created by an absorbed prior slice. External maps such as Ernst can orient attention, but they do not by themselves warrant opening a slice. A coverage gap becomes a warrant only when opened material has generated a question that needs that zone.

Before opening another pilot, run a quick absorption check: the prior pilot should have landed somewhere outside itself, such as the glossary, journal, page-map, citation map, an encounter, or a concrete thread. Record available next options with their warrants, not as a priority ranking.
