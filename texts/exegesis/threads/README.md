## Tagging Semantics

- `Threads` means the concept is already active in the entry as written, even if the later vocabulary has not stabilized yet
- `Foreshadows` means the entry clearly seeds a later concept, but Dick has not yet articulated that thread explicitly
- The table above records the first canonical or especially concentrated formulation for each thread, not every earlier activation or precursor

## Thread Rules

- A thread dossier records **key entries** where the thread appears, with brief notes on what's new or changed
- Note **mutations**: when does Zebra become VALIS? when does the Logos split from the H.S.?
- Note **contradictions**: when does Dick hold two incompatible positions simultaneously?
- Note **self-awareness**: does Dick know he's revising, or is he looping?
- Cross-reference other threads when they intersect (most entries touch 2–4 threads)

## Adding Threads

New threads may emerge during reading. If a pattern recurs across 5+ entries and isn't captured by existing threads, create a new dossier file and add it to this index.
