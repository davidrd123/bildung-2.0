# Zeitmauer Threads

Thread dossiers are the first-class commentary layer for `texts/zeitmauer`.

They are not replacements for:

- section-level translation notes in `parts/`
- glossary decisions in `source/glossary.yaml`
- session drift and method notes in `journal.md`

They sit between those layers. A thread gathers a recurring problem, image, or conceptual path into a reusable dossier that the viewer, notes, and agents can traverse directly.
