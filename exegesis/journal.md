# Reading Journal

## 2026-03-29 — Scaffold

Set up `exegesis/` as a separate project from `escolios` and `zeitmauer`. The key framing decision: this is an atlas, not a translation. The Exegesis is already in English; what it lacks is navigability. The meaning lives in recurrence and mutation across hundreds of entries, not in any single passage.

Design choices:

- Archival unit = the bracketed entry (`[folder:page]`), preserving the Jackson/Lethem/Williams numbering
- Interpretive unit = the thread (Zebra/VALIS, BIP, Thomas, orthogonal time, etc.)
- Reading passes follow chronological order because Dick's mode of thinking shifts dramatically across 1974–1982
- Thread dossiers are living documents that grow entry-by-entry, not pre-written summaries
- Source & Swerve (parallel texts with Dick's philosophical sources) deferred to Phase 2

Open questions at scaffold stage:

- How to handle the letters vs. the journal entries — they have different textures (letters are more structured, addressed to a reader; journal entries are more recursive and self-directed)
- Whether the ~33 letters should get their own entry type or be folded into the folder sequence
- How dense the per-entry annotation should be — the first pass on Folder 4 will calibrate this
- Whether "echoes" should be forward-only (pointing to later entries that revise this one) or bidirectional — probably bidirectional, added on re-reading passes

Current model of Dick (will change): a brilliant autodidact who experienced something genuine in 2-3-74 and spent eight years trying to find the conceptual framework adequate to it, never succeeding because the frameworks kept refracting the experience rather than containing it. The recursion isn't pathological — it's the honest trace of a mind that won't settle for an explanation that doesn't account for everything.
